# jwp-shopping-cart

## 기능 요구 사항

- [x] 상품 목록 페이지 연동
  - [x] / url로 접근할 경우 index.html을 통해 상품 목록 페이지를 조회 가능하다.
  - [x] 상품 기본 정보는 상품 ID, 상품 이름, 상품 이미지, 상품 가격이다.

- [x] 관리자 도구 페이지 상품 관리 CRUD API
  - [x] 생성(/admin/product POST)
  - [x] 조회(/admin)
  - [x] 수정(/admin/product/{id} PUT)
  - [x] 삭제(/admin/product/{id} DELETE)
