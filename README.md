# 📄 웹 장바구니 - 요구사항

# 도메인

## 😀유저

- [x] 이메일, 패스워드를 가진다.
- [ ] 이메일은 공백일 수 없다.
- [ ] 이메일은 이메일 형식이어야 한다.
- [ ] 패스워드는 공백일 수 없다.

## 🛒 장바구니

- [ ] 0개 이상의 상품을 가질 수 있다.
- [ ] 상품을 추가할 수 있다.
- [ ] 상품을 삭제할 수 있다.

## 🎁 상품

- [x]  ID, 이름, 이미지, 가격을 가진다.
- [x]  상품명은 100자 이하여야 한다.
- [x]  상품 가격은 0원 이상이어야 한다.
